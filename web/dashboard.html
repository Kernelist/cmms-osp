1단계: Feature 추출 (입력 데이터)

CMMS 이력 + 센서 신호에서 28개 feature를 추출하여 분류기에 입력합니다.

2단계: 분류기 (5개 독립 판정)

각 분류기가 독립적으로 RUL 등급을 판정합니다. 서로 다른 알고리즘을 사용하여 다양성을 확보합니다.

3단계: 가중치 도출 (Log-odds)

각 분류기의 테스트 정확도에서 Log-odds 변환으로 가중치를 자동 도출합니다.

Wi = log( Acci / (1 - Acci) )
정규화: Wi' = Wi / Σ Wj   (합 = 1.0)
Acc < 50% → 찬스 수준 이하 → 최소 가중치 부여

4단계: 앙상블 투표 (실시간 결과)

선택한 부품에 대해 각 분류기의 투표와 가중치 적용 과정을 시각화합니다.
